# rmw_dds_common: ROS 2 C++ utilities for DDS-based RMWs

`rmw_dds_common` is a collection of C++ APIs to support DDS based, C++ RMW implementations.

`rmw_dds_common`是 C ++ API 的集合，用于支持基于 DDS 的 C ++ RMW 实现。
